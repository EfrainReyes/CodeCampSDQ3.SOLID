CodeCampSDQ3.SOLID
==================

Projecto de ejemplo para Charla de SOLID Principles @ CodeCamp SDQ 3.0

======================================================================

Este projecto tiene como Objetivo principal mostrar como comunmente 
escribimos codigo violando principios de diseño de orientacion a 
objetos, y a su vez, como podemos corregir dichos errores de diseño
aplicando buenas practicas y patrones.

Enjoy!
